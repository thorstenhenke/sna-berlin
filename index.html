Vorbereitung auf den Workshop

Zur Vor- und Nachbereitung ist folgende Literatur empfohlen:

  • Snijders, T. A. B., van de Bunt, G. G., & Steglich, C. E. G. (2010). Introduction to stochastic actor-based models for network dynamics. Social Networks, 32(1), 44–60.

Software

R und RStudio

In allen drei Fällen werden wir zunächst R installieren und anschließend erst RStudio. Der Hintergrund ist der, dass es sich bei RStudio lediglich um eine grafische Oberfläche für R handelt. Wer mit einer anderen Oberfläche arbeiten möchte, kann demnach den zweiten Punkt überspringen.

Wer R und RStudio auf seinem Rechner installiert hat, sollte bitte vor dem Worksshop prüfen, ob es sich dabei auch um die aktuellste Version der Software handelt. Dies betriff vor allem die Version von R. Die Versionsnummer von R kann mit dem folgenden Befehl erfragt werden: 

R.version$version.string

Windows

Zunächst müssen die beiden aktuellsten Installationsroutinen heruntergeladen werden. Im Fall von RStudio benutzen wir die kostenlose OpenSource Variante für den Desktop. Zu finden sind diese unter den folgenden Links:

  1. https://cran.rstudio.com/bin/windows/base/
  2. https://www.rstudio.com/products/RStudio/#Desktop

Anschließend werden beide Programme in der folgenden Reihenfolge installiert (Doppelklick um den Installationsvorgang zu starten):

  1. R
  2. RStudio

Um zu kontrollieren, ob die Installation funktioniert hat, öffnet RStudio einmal. In der so genannten Konsole/Terminal/Eingabeaufforderung sollte eine lange Meldung erscheinen, die euch über die verwendete Version von R und die Lizensierung von R informiert.

Mac OSX

Zunächst müssen die beiden aktuellsten Installationsroutinen heruntergeladen werden. Im Fall von RStudio benutzen wir die kostenlose OpenSource Variante für den Desktop. Zu finden sind diese unter den folgenden Links:

  1. https://cran.rstudio.com/bin/macosx/. Wählen Sie hier bitte das erste angebotene Paket mit dem Namen R-<VERSIONSNUMMER>.pkg aus. Alle weiteren Optionen (z.B. -snowleopard.pkg sollten ignoriert werden).
  2. https://www.rstudio.com/products/RStudio/#Desktop

Anschließend werden beide Programme in der folgenden Reihenfolge installiert (Doppelklick um den Installationsvorgang zu starten):

  1. R
  2. RStudio

Um zu kontrollieren, ob die Installation funktioniert hat, öffnen Sie RStudio bitte einmal. In der so genannten Konsole/Terminal/Eingabeaufforderung sollte eine lange Meldung erscheinen, die Sie über die verwendete Version von R und die Lizensierung von R informiert.

Linux

In der Folge wird die Installation für Ubuntu beschrieben. Die Installation auf anderen Distributionen kann unter umständen abweichen. In aller Regel unterhalten die verschiedenen Linix-Distributionen jedoch eigene Wikis, in denen sich zumeist auch ein Eintrag zu R befindet. 

sudo apt-get install r-base
cd ~/Downloads
wget https://download1.rstudio.org/rstudio-0.99.489-i386.deb
sudo dpkg -i ~/Downloads/rstudio-0.99.489-i386.deb

R-Pakete

Für den Workshop werden wir zwei verschiedene Pakete von R-Siena benötigen. 

install.packages(c("RSiena", "RSienaTest"), repos="http://R-forge.R-project.org")

UCinet und NetDraw

Sowohl UCinet als auch Netdraw sind ausschließlich für Windows erhältlich. Mac OSX sowie Linux Nutzer müssen entweder auf einen Emulator (z.B. Wine) oder eine Virtualisierungsumgebung (z.B. VirtualBox, Parallels, etc.) zurückgreifen. Da Netdraw in UCinet enthalten ist, wird nachfolgend lediglich die Installation von UCinet beschrieben. Bitte beachtet, dass es sich hierbei um ein kommerzielles Softwarepaket handelt und die Nutzung von UCinet nur für die Testphase von 60 Tagen kostenlos möglich ist.

  1. Download 32bit Testversion. Bitte wählt unter den angebotenen Programmversionen die 32-Bit Version aus, auch wenn ihr ein 64-Bit Betriebssystem habt. Die beiden angebotenen Versionen unterscheiden sich im Funktionsumfang.
  2. Installation mit Doppelklick auf Setup32UCI6596.exe

Kurzberschreibung

Der Workshop soll Forschenden der Erziehungswissenschaften, der Psychologie und den anderen Fachdidaktiken einen Einblick in die Erhebungs- und Analysemethoden der sozialen Netzwerkforschung geben sowie in die Arbeit mit vornehmlich längsschnittlichen soziometrischen Daten einführen. Es wird sowohl eine Einführung in Konzepte und Methoden der sozialen Netzwerkanalyse gegeben (Lysann Zander, Freie Universität Berlin) als auch eine Einführung in die RSiena-Software (Thorsten Henke, Universität Potsdam). Darüber hinaus erfolgt eine weiterführende Einweisung in die Arbeit mit längsschnittlichen Netzwerkdaten (Tobias Stark, Universität Utrecht).
